UCPD Incident Tracking

This project was undertaken with the goal of showing a more granular picture of incidents posted on the UCPD Incident Report page.

The current analysis of the incident data is listed below:

  • A map of incidents that took place over the last 30 days: Link
  • A drilldown bar chart that allows you to see the type of events that happen at a given hour in a given season based on of data from 2011 to the most recently completed year: Link
  • A bar chart showing all incident types that occurred more than 20 times within the last 365 days: Link

The code for this project is stored in two places:

  • The UCPD incident web scraper: Link
  • The FastAPI application for data display: Link
